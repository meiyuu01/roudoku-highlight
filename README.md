# 朗読ハイライト

[▶ アプリを開く](https://roudoku-highlight.pages.dev/)

朗読・音読・読書のためのシンプルなWebアプリ。  
原稿を区切りごとにハイライトしながら読み進められます。

## 特徴

- **段落・文・読点ごとに区切って表示**：今読んでいる場所がはっきり分かる
- **読み終わった部分はグレーアウト**：進捗が一目で見える
- **ルビ対応**：青空文庫互換の `漢字《かんじ》` 記法
- **縦書き／横書き切り替え**
- **ダークモード**
- **読み上げ機能**（ブラウザの音声合成を利用）
- **オフライン動作**（PWA対応）
- **自動保存**：書きかけの原稿はブラウザに保存されます

## 使い方

1. 左側の「原稿」欄にテキストを貼り付け
2. ルビを振りたい部分を選択して「《ルビ挿入》」ボタン（または `Ctrl+Enter`）
3. 「▶ 再生モード」を押して朗読モードへ

### 再生モード操作

| 操作 | キー／ジェスチャー |
|------|------------------|
| 次へ | `Space` / `Enter` / `↓` / `→` / 右スワイプ |
| 戻る | `↑` / `←` / 左スワイプ |
| 終了 | `Esc` |
| ジャンプ | 文をクリック／タップ |

### ルビの書き方

- `漢字《かんじ》` … 漢字部分にルビ
- `｜任意の範囲《ふりがな》` … ひらがなを含む範囲にもルビ

## インストール

ブラウザの「インストール」または「ホーム画面に追加」で、アプリとして使えます（PWA）。  
インストール後はオフラインでも動作します。
